=====================================================
TEACHER ASSIGNMENT FEATURE - QUICK TEST SETUP
=====================================================

Run these SQL scripts IN ORDER in Railway PostgreSQL Query console:

STEP 1: Create Sections and Sample Classes
-------------------------------------------
File: backend/seed-sections-and-subjects.sql
What it does:
- Creates Section A for all grade levels
- Creates 5 sample subjects for Grade 1 Section A:
  * English
  * Mathematics  
  * Science
  * Filipino
  * MAPEH

STEP 2: Create Test Teachers with Assignments
----------------------------------------------
File: backend/seed-test-teachers-and-assignments.sql
What it does:
- Creates 3 test teachers:
  * Ms. Sarah Johnson (English, Filipino specialist)
  * Mr. John Smith (Math, Science specialist)
  * Ms. Maria Santos (MAPEH specialist)
  
- Assigns 2 teachers to Grade 1 subjects:
  * Ms. Sarah Johnson → English (Grade 1)
  * Mr. John Smith → Mathematics (Grade 1)

STEP 3: Test in the Application
--------------------------------
1. Login to your app
2. Go to: Grade Levels → Grade 1
3. You should see 5 subject cards:
   ✅ English - Shows "Ms. Sarah Johnson" with Remove button
   ✅ Mathematics - Shows "Mr. John Smith" with Remove button
   ⭕ Science - Shows "+ Assign Teacher" button
   ⭕ Filipino - Shows "+ Assign Teacher" button
   ⭕ MAPEH - Shows "+ Assign Teacher" button

4. Test Assigning:
   - Click "+ Assign Teacher" on Science
   - Should show "Mr. John Smith" (qualified for Science)
   - Click to assign
   
5. Test Removing:
   - Click "Remove" on English
   - Confirms removal
   - Button changes back to "+ Assign Teacher"

=====================================================
TEACHER QUALIFICATIONS (for testing):
=====================================================
Ms. Sarah Johnson can teach:
  ✓ English
  ✓ Filipino

Mr. John Smith can teach:
  ✓ Mathematics
  ✓ Science

Ms. Maria Santos can teach:
  ✓ MAPEH

NOTE: When you click "+ Assign Teacher" on a subject,
only teachers qualified for that subject will appear
in the selection modal.

=====================================================
TROUBLESHOOTING:
=====================================================
If you don't see subject cards after running scripts:
1. Hard refresh browser (Ctrl+Shift+R)
2. Check Railway logs for migration success
3. Verify scripts ran without errors in Railway Query

If "+ Assign Teacher" doesn't show teachers:
- Make sure teachers have the subject in teacher_subjects
- This is intentional - only qualified teachers can be assigned
